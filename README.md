# 🔐 Login & Registration System (C++)

A simple console-based Login & Registration System developed in C++ using file handling. This project allows users to create accounts, store credentials, and securely log in with validation checks.

---

## 📌 Project Overview

This system provides basic authentication functionality by allowing users to register with a username and password, and later log in using those credentials. User data is stored in a file to maintain persistence.

---

## 🚀 Features

- 📝 User Registration
- 🔍 Duplicate Username Check
- 🔐 Password Confirmation Validation
- 📏 Minimum Password Length Check
- 🔑 Secure Login System
- 🚫 Limited Login Attempts (3 tries)
- 💾 File Handling for Data Storage

---

## ▶️ How to Run

1. Open the code in any C++ compiler (Dev C++, VS Code, etc.)
2. Compile the program
3. Run the executable
4. Choose an option:
   - Register a new account
   - Login with existing credentials
5. Follow on-screen instructions
